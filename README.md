## 📌 Overview

This graduation project is a **Smart Health Monitoring System** designed to provide **real-time, non-invasive health measurements** using embedded sensors and cloud connectivity.

The system integrates:

- 📟 **ESP32 microcontroller**
- 🌡️ **Body temperature monitoring**
- 🩸 **Non-invasive glucose estimation**
- ☁️ **MongoDB database**
- 📱 **Mobile application for monitoring**
- 🔔 **Alerts and notifications**

The goal is to build a **portable IoT healthcare device** capable of collecting physiological data, uploading it to the cloud, and displaying it through a user-friendly mobile interface.

---

## 🚀 Features

✅ Real-time sensor data acquisition  
✅ Body temperature monitoring  
✅ Non-invasive glucose level estimation  
✅ OLED display interface  
✅ Cloud data synchronization using MongoDB  
✅ Mobile app dashboard  
✅ Alert system using buzzer notifications
